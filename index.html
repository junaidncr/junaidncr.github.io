Projects

BMI Calculator App

Developed in my 6th-semester project during my BS-IT degree, the BMI Calculator app is a simple yet effective tool designed to calculate Body Mass Index (BMI) based on user input. With a clean and intuitive user interface, the app allows users to input their height and weight, performs the necessary calculations, and provides instant feedback on their BMI status. By leveraging basic Android development concepts such as user input handling, calculation algorithms, and UI design principles, this project served as a valuable learning experience in mobile app development. Through the implementation of simple colors, buttons, and input fields, I gained hands-on experience in creating functional and user-friendly Android applications.

Grade Evaluator App

The Grade Evaluator app, part of my coursework during my BS-IT degree, is a comprehensive tool designed to assist students in assessing their academic performance across multiple courses. With a user-friendly interface, the app allows users to input their obtained marks for up to five course subjects and calculates the total obtained marks along with the corresponding grade based on predefined criteria. Additionally, the app provides personalized remarks to offer constructive feedback to the user, such as encouragement to work harder or acknowledgment of a job well done. By integrating total marks calculation, grade assignment, and dynamic feedback generation, this project provided valuable insights into Android app development and user interaction design.
